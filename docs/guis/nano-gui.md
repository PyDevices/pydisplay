# Nano-GUI

[Nano-GUI](https://github.com/peterhinch/micropython-nano-gui) by Peter Hinch — lightweight GUI for memory-constrained MicroPython boards.

## Config

Requires `color_setup.py` in your project (copy from [`src/add_ons/color_setup.py`](https://github.com/PyDevices/pydisplay/blob/main/src/add_ons/color_setup.py)). Wire it to pydisplay's display and fonts.

## Example

`src/examples/nano_gui_simpletest.py`

## Platform

MicroPython only.
