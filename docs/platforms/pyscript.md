# PyScript

Run pydisplay in the browser via [PyScript](https://pyscript.net/) and `displaysys.psdisplay.PSDisplay`.

!!! warning "Work in progress"
    PyScript support is experimental. Only touchscreen input is implemented. Most examples use blocking loops and **will not run** in the browser.

## Live demo

- **Online:** [PyDevices.github.io/pydisplay/demo/](https://PyDevices.github.io/pydisplay/demo/)
- **Local:** clone the repo, run from the **repository root** (not `src/`):

```bash
python -m http.server
```

Open [http://localhost:8000/](http://localhost:8000/) — the demo hub links to each HTML page under `html/`.

## Demo pages

| Page | Purpose |
|------|---------|
| `html/example.html?script=calculator` | Run one example by name |
| `html/test.html` | Example picker + terminal |
| `html/repl.html` | REPL with canvas |
| `html/editor.html` | mpy-editor with `paint.py` |

Manifest: `html/pyscript.toml` (regenerated by `./tools/regenerate.sh`).

## asyncio requirement

PyScript runs on asyncio. Examples need `async def`, `await`, and `asyncio.get_event_loop().run_until_complete(main())` (or equivalent). Blocking `while True:` loops without `await` will hang the browser tab.

## PyScript-compatible examples

Only these examples in `src/examples/` use asyncio today:

| Script | Notes |
|--------|-------|
| `calculator.py` | Best starting point; linked from demo hub |
| `paint.py` | Used by `html/editor.html` |
| `eventsys_simpletest.py` | Minimal event loop test |
| `apollo.py` | Apollo DSKY demo; heavier |

Try others via `html/example.html?script=<name>` — most will fail until ported to asyncio.
