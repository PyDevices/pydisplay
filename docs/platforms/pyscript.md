# PyScript

Run pydisplay in the browser via [PyScript](https://pyscript.net/) and `displaysys.psdisplay.PSDisplay`.

!!! warning "Work in progress"
    PyScript support is experimental. Only touchscreen input is implemented. Most examples are not asyncio-compatible and will not run in the browser.

## Live demo

- **Online:** [PyDevices.github.io/pydisplay/demo/](https://PyDevices.github.io/pydisplay/demo/)
- **Local:** clone the repo, run from the **repository root** (not `src/`):

```bash
python -m http.server
```

Open [http://localhost:8000/demo/](http://localhost:8000/demo/) after the demo deploy workflow is active, or [http://localhost:8000/](http://localhost:8000/) for the landing page.

## Demo pages

| Page | Purpose |
|------|---------|
| `html/example.html?script=calculator` | Run one example by name |
| `html/test.html` | Example picker + terminal |
| `html/repl.html` | REPL with canvas |
| `html/editor.html` | mpy-editor with paint.py |

Manifest: `html/pyscript.toml` (regenerated by `./tools/regenerate.sh`).

## asyncio requirement

PyScript runs on asyncio. Examples must use `async def` and `await`, or avoid blocking loops. **`calculator.py`** is the reference asyncio example.
