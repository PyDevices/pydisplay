How to use

  • ?modules= — comma-separated top-level module stems (no .py).
  • ?folders= — comma-separated folder examples; each installs html/<name>.json via MIP.
  • Both may appear; whichever param comes first in the URL supplies the import entry point.
  • Click Run to install everything, then import the entry.
