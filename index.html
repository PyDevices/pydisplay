PyDevices / pydisplay

Cross-platform display, input & graphics — one Python codebase.

pydisplay is the foundation of the PyDevices stack: pure-Python display, input, and event drivers that run unmodified on MicroPython, CircuitPython, and CPython. The same examples and board configs stretch from MCU panels to desktop windows and the browser — without a rewrite when you change interpreter.

Start with the live PyScript demos, dig into the docs for install paths (MIP, GitHub packages, TestPyPI), and add native cmods or LVGL when you need more speed or a full widget toolkit. The portable layer stays Python so your application logic stays shareable.
