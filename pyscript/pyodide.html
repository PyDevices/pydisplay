Pyodide

Dev tool: run examples under Pyodide with ?modules=name and/or ?manifests=name (same MIP JSON as micropython.html). Also supports ?deps= (micropip from TestPyPI / PyPI). Whichever of modules/manifests comes first supplies the import entry.

How to use

  • ?modules= — comma-separated top-level module stems (no .py).
  • ?manifests= — comma-separated MIP manifest names; each installs packages/<name>.json (same files as the MicroPython gallery).
  • Both may appear; whichever param comes first in the URL supplies the import entry point.
  • ?deps= — comma-separated PyPI / TestPyPI project names (or direct .whl URLs) installed via micropip.
  • Click Run to install, then import the entry.
