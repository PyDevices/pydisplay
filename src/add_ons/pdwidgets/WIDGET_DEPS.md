## Always-on core (every UI)

Ship these regardless of which widgets you use:

- `widget`, `display`, `screen`, `task`, `_constants`, `_themes`, `_util`, `_focus`

Everything under `widgets/` is optional cargo. Prefer lean imports:

```python
from pdwidgets.widgets.button import Button
```

Convenience `import pdwidgets as pd` resolves names lazily via `__getattr__`
(loads a widget module on first access). Icons under `icons/` are optional —
omit them if no `IconTheme` / `.pbm` assets are used.
