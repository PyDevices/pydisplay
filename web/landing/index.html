PyDevices / pydisplay

Display, input & events — one Python codebase.

pydisplay is the foundation of the PyDevices stack: pure-Python display drivers, input, and a portable Runtime that run unmodified on MicroPython, CircuitPython, and CPython. The same board configs and examples stretch from MCU panels to desktop windows, the browser, and Android.

The biggest on-ramp is the installable PyScript PWA — live demos in the browser with no flash step. Need a native phone build? The pydisplay_android companion ships a python-for-android APK path for the same stack.

Core runtime

board_config + Runtime — not a separate event broker 
from board_config import display_drv, runtime

runtime.on(runtime.events.MOUSEBUTTONDOWN, on_click)
runtime.run_forever()

board_config wires display_drv and runtime for your host. Register handlers with runtime.on(...), then drive the loop with runtime.run_forever(). The same idiom appears in MCU examples, the PyScript gallery, and the Android paint demo.
