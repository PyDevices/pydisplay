pydisplay demos

These gallery demos run the same pydisplay examples you would load on MicroPython, CircuitPython, or CPython — here inside the browser with PyScript. Use them to explore drawing, input, widgets, and LVGL without flashing a board first.

Open a card and click Run. The first load may take a few seconds while the runtime downloads; featured demos stay pinned at the top. Each card has MicroPython and Pyodide links with the matching ?deps= install query (MIP vs micropip).
