Loader

Load with ?modules=name and/or ?manifests=name (e.g. ?modules=calc_graphics,calc_engine, ?manifests=chango).

How to use

  • ?modules= — comma-separated top-level module stems (no .py).
  • ?manifests= — comma-separated MIP manifest names; each installs packages/<name>.json.
  • ?deps= — comma-separated packages installed via MIP (micropython-lib index).
  • Modules/manifests may both appear; whichever param comes first in the URL supplies the import entry point.
  • Click Run to install everything, then import the entry.
