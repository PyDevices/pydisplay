Pyodide

Dev tool: run examples under Pyodide with ?modules=name and/or ?manifests=name (same MIP JSON as micropython.html). LVGL demos micropip-install the pyemscripten_2026_0 wheel (local ./wheels/ or lv_cpython_mod Pages). Whichever of modules/manifests comes first supplies the import entry.

How to use

  • ?modules= — comma-separated top-level module stems (no .py).
  • ?manifests= — comma-separated MIP manifest names; each installs <name>.json from this directory (same files as the MicroPython gallery).
  • Both may appear; whichever param comes first in the URL supplies the import entry point.
  • ?packages= is not supported — use micropython.html for that.
  • # pyodide wheels: lvgl in the entry example → micropip-install via wheels/lvgl.json (lv_cpython_mod Pages or local serve.py); override with ?lvgl_wheel=.
  • Click Run to install, then import the entry.
